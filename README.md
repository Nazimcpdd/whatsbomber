<h3>Installing</h3>

```java
$ apt update && apt upgrade

$ apt install curl
$ apt install figlet
$ apt install ruby
$ gem install lolcat
$ git clone https://github.com/ArroKM/VIRTEX
$ cd VIRTEX
$ chmod +x virtex.sh
$ sh virtex.sh
```
